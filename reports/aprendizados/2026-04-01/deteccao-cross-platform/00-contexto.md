# Contexto do Estudo

## Objetivo
Descobrir como fazer detecção automática de ambiente (SO, arquitetura, modo de execução) e auto-dispatch de agentes de forma que funcione em QUALQUER computador sem configuração manual.

## Escopo
- Detecção de SO: macOS, Windows (Git Bash, PowerShell, cmd), Linux (X11, Wayland), WSL
- Detecção de arquitetura: Intel, Apple Silicon, ARM Linux
- Detecção de modo: terminal CLI vs extensão VS Code
- Auto-dispatch: abrir nova janela/aba e colar prompt em cada plataforma

## Modo Escolhido
Arquitetural-Técnico — o comandante precisa implementar uma solução definitiva

## Limitações de Ambiente
- Acesso web total disponível
- Ferramentas de busca e leitura de arquivo disponíveis
- Não é possível testar em Windows/Linux diretamente (apenas macOS disponível)
