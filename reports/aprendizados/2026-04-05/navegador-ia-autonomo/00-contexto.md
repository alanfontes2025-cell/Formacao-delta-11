# Contexto da Pesquisa

## Objetivo
Encontrar a melhor ferramenta de navegador para que IAs (Claude Code, Codex) possam acessar a internet de forma autonoma — navegar sites, clicar, preencher formularios, tirar prints, ver codigo de paginas — sem intervencao humana.

## Escopo
- macOS ARM64 (Apple Silicon, MacBook Pro)
- Conexao via MCP (Model Context Protocol) com Claude Code
- Prioridade 1: funcionalidade completa
- Prioridade 2: leveza (pouca memoria/processador)
- Prioridade 3: funcionar como padrao global para todos os projetos

## Ambiente do Usuario
- Chrome 146 e Firefox 149 instalados
- Puppeteer MCP configurado (mas descontinuado — descoberto na pesquisa)
- Playwright disponivel via Python (cache UV)
- Node.js v22.22.1 (ARM64 nativo via nvm)

## Limitacoes da Pesquisa
- Acesso web: total (WebSearch + WebFetch)
- Nivel de citacao: links verificaveis
- Ferramentas: disponiveis
- Memoria persistente: disponivel

## Metodologia
Estudador — Arquiteto de Certezas Universal 7.0
- 36-55 fontes estimadas
- Hierarquia: Ouro > Prata > Bronze > Ferro > Chumbo
- Triangulacao com modo contradicao para afirmacoes criticas
